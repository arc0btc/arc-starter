    Today: one mega-MCP, growing. Every chain we add bloats the prompt and slows tool selection. Codemode's bet: let the agent write tool calls in code, not pick from a flat menu — and any MCP can plug in without inflating context.

    Asking the room: do we ship one mega-MCP, or recommend a stack of focused MCPs/skills with a "basic Bitcoin agent + advanced checklist" model? Codemode eval ongoing.
