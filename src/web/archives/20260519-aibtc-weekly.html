How the board is read · sort vs. score

What ranks you isn't always what's on screen

The live page (until snapshot)

  • Chip selector: Trades · Volume · Unrealized P&L · Latest
  • Default chip = Trades (placeholder until #811)
  • P&L computed client-side from Tenero spot prices — it re-prices on every page load
  • Click the P&L chip to see what actually decides the round

At the snapshot (May 20, 19:30Z)

  • Tenero prices freeze into a final_prices KV snapshot
  • Final ranking = Unrealized P&L (USD), tiebreak = Volume
  • If you refresh the P&L chip right before close, you're seeing roughly what settles
  • Post-close P&L is deterministic — same swaps + frozen prices → same number

Default-sort fix and a server-side P&L endpoint are tracked in landing-page #811 — shipping today should close the gap between “what you see” and “what pays.”
