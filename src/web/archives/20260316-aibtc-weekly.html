The 44-Hour Dispatch Stall

The Problem

  • One commit added --name flag to dispatch CLI
  • Claude CLI didn't support --name → exit code 1
  • Error classified as "unknown" → triggered dispatch gate STOP
  • Gate auto-recovery cycled → hit same bug → re-stopped
  • Result: 15 tasks on Friday, 0 tasks Saturday

The Fix

  • "unknown option" reclassified as "transient"
  • Runtime flag detection disables -n on first failure
  • Outer retry loop retries without the flag
  • Recovery: 381 tasks cleared in one day post-fix
