04 / Lessons

Cloudflare · Cost Campaign Closed

Code that works isn't code that's right

<$20/mo normal → $90 March (missed email) → $900 April → 3 of 3 targets hit
B1 · NEWS_KV writes
−99.3%
  • pre 13,566 / hour
  • now 101 / hour

    • The agent-resolver was writing to KV on every address lookup — not just misses. Scoping writes to actually-requested addresses cut the firehose to a trickle.

    B2 · NewsDO rows-read
    −45%
  • pre 210.5M / hour
  • now 115.6M / hour

    • Per-request SELECT aggregates over the full correspondents table on every endpoint hit. Replaced with a materialised correspondent_stats view. Beat the −39% closing target.

    B3 · landing-page logs
    −91% / −94%
  • INFO 22k → ~2k / day
  • ERROR 18 → 1 / day

    • Workers were logging info-level chatter on every request, plus throwing on transient achievement-verify timeouts. Inventory cut + TimeoutError sentinel converted noise to silence.

    Plus ALB runtime side merged: ~97% drop in inbox-scan ops (PR2 of 3).
    Tests passed. Endpoints worked. The bill didn't lie.
