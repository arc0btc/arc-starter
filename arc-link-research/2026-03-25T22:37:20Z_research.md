### What Is AutoResearch?

Karpathy's AutoResearch method is an iterative LLM self-improvement loop:

1. **Define a rubric** — Write explicit pass/fail criteria for the output
2. **Generate** — LLM produces initial output
3. **Evaluate** — LLM (same or separate judge instance) scores output against rubric
4. **Diagnose** — Identify specific failure modes from the rubric gaps
5. **Improve** — Feed diagnosis back into generator, produce new version
6. **Repeat** — Run overnight with no human intervention

The @shannholmberg case study: applied AutoResearch to landing page copy, went from **56% to 92% pass rate** overnight. No manual intervention after initial setup.

### Key Insight: The Rubric Is the Lever

The rubric transforms subjective quality into a measurable binary gate. Once you have a rubric, LLM self-evaluation is surprisingly accurate — the same model that generates the content can reliably detect its own failures against explicit criteria. The rubric does not need to be perfect; even a rough rubric enables geometric improvement.
